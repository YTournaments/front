# Ytournaments

Créer l'impulsion chez les élèves pour devenir les prochains acteurs de l'évènementiel compétitif, voilà le défi que s'est donné l'équipe de YTournaments !

![Logo](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/th5xamgrr6se0x5ro4g6.png)

## Installation

Installation

Si vous n'avez pas pnpm voici la commande:

```bash
npm install -g pnpm
```

```bash
  cd front
  pnpm install

```

## Lancer en local

Clone the project

```bash
  git clone https://github.com/AnthonY-Dole/YTournamentsFront.git
```

Go to the project directory

```bash
  cd front
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  npm run dev
```

## Environment Variables

Pour exécuter ce projet, vous devrez ajouter les variables d'environnement suivantes à votre fichier .env
Créer le fichier à la racine du projet si vous ne l'avez pas

`API_URL:`

`API_KEY:`
